# Saving Cycle / Tanda

***

A web app that helps organize tandas and notifies participants when the due day of their payment is approaching, helping everyone get their money on time. [Tanda](https://www.thebalance.com/how-do-tandas-work-4582180) is a Latin American term for an informal, short-term loan among friend or family members.

**Tools** Node, Express, PostgreSQL, React
